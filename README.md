# antidepressant-biomarker-meta-analysis
 dissertation using meta-analysis, investigating inflammatory biomarkers and antidepressant response on patient with MDD.
This project investigates whether baseline inflammatory biomarkers (CRP, IL-6 and TNF-α) predict antidepressant treatment response in Major Depressive Disorder.

Methods:
- Systematic review
- Effect size conversion
- Random-effects meta-analysis
- Heterogeneity analysis
- Funnel plots
- Leave-one-out sensitivity analysis

Tools:
- R
- metafor
- meta
- Google Colab
